Coding

There are many great, free resources you can use to learn how to code. You can use these if you're a complete beginner or someone who already has experience.

Scratch

Scratch is a block-based programming website that is perfect for beginners. You can use their block system to create games and get used to the debugging process.

The Odin Project

The Odin Project is a free course that teaches web development. It is beginner-friendly and provides step-by-step instructions on projects.

Free Code Camp

Free Code Camp is similar to The Odin Project, but it offers many more courses. In addition to web developmet, it teaches Python, databases, and back-end development. The courses range in difficulty.

Code.org

Code.org is a site that gives users the chance to explore coding on their own. It offers block-based and text-based coding, making it great for all experience levels. It also offers courses on areas like AI.

Codecademy

Codecademy offers many courses for many different langauges. The courses vary in length and difficulty, with some being beginner-friendly and some being challenging even for experienced coders. Some of the courses are also not free.
